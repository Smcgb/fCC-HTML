fCC Responsive Web Design

Introduction

freeCodeCamp is a community of people from all around the world who are learning to code together.

In the Responsive Web Design Certification (RWD), you'll learn the languages that developers use to build webpages: HTML (Hypertext Markup Language) for content, and CSS (Cascading Style Sheets) for design.

The RDW Certification has many projects to teach the necessary concepts for web development such as:

  • Survey form, introductory HTML and CSS
  • Tribute page, CSS box model, accessibility and typography
  • Technical Documentation: Intermediate CSS, Intro to Responsive Design, CSS Pseudo Selectors
  • Product Landing Page, CSS Variables and Grid
  • Personal Portfolio Project to put it all together

Survey Form

In this project, you will build a survey form to collect data from users. This project introduces you to basic HTML elements and form inputs, as well as basic CSS styling.

Skills you will learn:

  • HTML form elements
  • Basic CSS styling
<form>
<label for="name">Name:</label>
<input type="text" id="name" name="name">
</form>

Tribute Page

This project involves creating a tribute page to a person of your choice. You will learn about the CSS box model, accessibility, and typography principles.

Skills you will learn:

  • CSS box model
  • Accessibility practices
  • Typography
<div class="container">
<h1>Tribute to [Person]</h1>
<p>[Biography]</p>
</div>

Technical Documentation

In this project, you will create a technical documentation page. This will help you understand intermediate CSS concepts, responsive design, and CSS pseudo selectors.

Skills you will learn:

  • Intermediate CSS
  • Responsive design principles
  • CSS pseudo selectors
<article>
<h2>CSS Selectors</h2>
<p>CSS selectors are used to select the content you want to style.</p>
</article>

Product Landing Page
